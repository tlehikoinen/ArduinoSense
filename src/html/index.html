Arduino Sense Backend Documentation

This API is used to save temperature and humidity data

/user

method url body returns
GET / Users
GET /:id User with ID
POST /initialise Creates user table (if it does not exist)
POST / username, password Status code 200 if user was created successfully
POST /login username, password JSONWEBTOKEN if username and password were correct

/data

method params body Authentication returns
GET / All data
GET /:username Data for user
POST /initialise Creates data table (if it does not exist)
POST / temperature, humidity Bearer ${token_from_login} Status code 200 if data was saved successfully
